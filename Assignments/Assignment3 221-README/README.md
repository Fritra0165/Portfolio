# Overview
This program counts the number of text files in a given folder.

# Code Overview
## Purpose
The objective of this Java class is to recursively find the number of .txt files in a given directory.

## Key Methods
- `countTextFiles(File folder)
	- Iterates through the given files and folder in the directory folder 
	- users recursion to go deeper into the file structure 
	- prints out every single .txt file found in the process
	- returns total count 
- main(string[] args)
	- runs countTextFiles method with a file path

## Pre-requisites 
- Assumes the .java file is loaded into an IDE or in a folder where `FolderCreator.java` file is already present and executed to create a dummy structure of folder starting at mainFolder
