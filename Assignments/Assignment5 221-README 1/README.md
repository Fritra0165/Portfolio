# Custom Type System and inference engine
---
## Description
This project simulates a type system. It includes types like primitives, arrays, and generics. It supports compatibility checks, type inference, and a key-value type-nameho collection.

## Features

- Abstract Type class with the methods:
	* PrimitiveType
	* ArrayType 
	* GenericType`
- TypeSafeCollection<K, V>: with compatibility check
- TypeInferenceEngine: for registering and retrieving inferred types
- Custom TypeMismatchException: for invalid operations
## Technologies Used

- Java
- HashMap, Iterator
- Inheritance and Interfaces
- Exception Handling
