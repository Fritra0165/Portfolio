# Overview
This program converts various units of measurement, such as length, weight, temperature, currency, speed, and volume.

# Code Overview
## Features 
Units of conversion the program supports 
+ Length: meters, feet, kilometers, miles
+ Weight: kilograms, pounds, grams, ounces
+ Temperature: Celsius, Fahrenheit, Kelvin
+ Currency: USD, EUR, Bitcoin (fixed exchange rate)
+ Speed: km/h, mph
+ Volume: liters, gallons
## Usage
Follow on-screen prompts:
+ convert (unit_from)(unit_to)(value)
+ Ex: convert meters feet 10
Type 'exit' to quit the program.
